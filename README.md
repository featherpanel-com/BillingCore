# BillingCore

Powerful billing management core for FeatherPanel. Provides user credit balances, billing profiles, currency settings, and invoice management that other billing addons build on.


## Features

- **User-facing**
  - **Billing → Billing Information** — view credits balance and manage billing profile
  - **Billing → My Invoices** — list and view invoice details

- **Admin**
  - **Billing Core** — manage credits (add/remove/set), search users, view statistics
  - **Currency settings** — default currency, available currencies list
  - **Credits mode** — `currency` (paid hosting) or `token` (freemium); tokens-per-currency rate
  - **Billing profiles** — admin and per-user billing info
  - **Invoice Management** — create, edit, delete invoices and line items
  - **Admin user widget** — credits, billing profile, and invoice history on user edit page
